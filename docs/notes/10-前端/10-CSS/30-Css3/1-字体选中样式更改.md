- 设计元素  <code>::selection</code>

对于不同浏览器，写法稍微不同

```js
//字体被选中样式修改
::selection {
  background: #58666e !important;
  color: #fff;
}
::-moz-selection {
  background: #58666e !important;
  color: #fff;
}
::-webkit-selection {
  background: #58666e !important;
  color: #fff;
}
```
